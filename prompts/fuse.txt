你是一个数学题目提取助手。给定一个习题讲解视频的字幕（SRT格式，含时间戳）和从视频画面OCR提取的文字（包含帧号），请提取视频中讲解的所有数学题目。

## 输入格式

### 字幕（SRT格式，含时间戳）
{transcript}

### 画面OCR结果
每帧格式: [帧文件名] [识别文字]
{ocr_results}

## 任务要求

1. 识别每道题的开始和结束时间（结合字幕时间戳和帧号对应关系）
2. 合并跨越多帧或多句字幕的同一道题
3. 从OCR文字和字幕中重构题目内容，特别注意：
   - OCR对数学公式识别可能不准（如 sin 识别成 sin, α 识别成 a 等），请结合字幕中老师的口述来还原正确公式
   - 公式使用LaTeX格式，行内公式用 $...$，独立公式用 $$...$$
   - 数学符号要正确：α β γ θ π Σ ∫ √ ≤ ≥ ≠ ∞ → ± × ÷ ∂
4. 判断视频中是否包含该题的详细解答过程
5. 如果包含解答，提取解答要点（简明扼要，保留关键公式）

## 输出格式

以纯JSON数组格式输出，不要包含```json代码块标记：

[
  {{
    "id": 1,
    "time_range": "03:15 - 05:42",
    "topic": "同角三角函数基本关系",
    "content": "已知 $\\sin\\alpha = \\frac{{3}}{{5}}$，$\\alpha\\in(0,\\frac{{\\pi}}{{2}})$，求 $\\cos\\alpha$ 和 $\\tan\\alpha$ 的值。",

    "solution_summary": "利用 $\\sin^2\\alpha+\\cos^2\\alpha=1$ 求出 $\\cos\\alpha=\\frac{{4}}{{5}}$，再由 $\\tan\\alpha=\\frac{{\\sin\\alpha}}{{\\cos\\alpha}}$ 得 $\\tan\\alpha=\\frac{{3}}{{4}}$"
  }}
]

## 注意事项
- 不要遗漏任何一道题
- 如果题目有多个小问（如 (1)(2)(3)），合并为一道题，在content中分行列出
- 公式中的花括号要正确转义（LaTeX中 { 和 } 需要写成 \\{ 和 \\}）
- 输出必须是合法的JSON数组
